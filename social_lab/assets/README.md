# assets/

Cartella per materiali grezzi (foto, video, screenshot).

**Questa cartella e' gitignored** — i file pesanti non finiscono nel repo. Tienila in locale o sincronizzala tramite cloud separato (Dropbox, iCloud, Drive).

## Struttura

```
screenshots_app/   screenshot dell'app (home, registrazione, classifica, Lifeline, report, ecc.)
paesaggi/          foto e video di montagna, natura, sentieri (B-roll per reel)
video_demo/        screen recording che mostrano feature in azione
ambassador/        foto/video di utenti reali che accettano di essere ricondivisi
generated/         asset generati (template grafici, card statistiche, meme)
```

## Convenzioni di naming

Formato: `YYYY-MM-DD_<tema>_<dettaglio>.<ext>`

Esempi:
- `2026-04-22_screenshot_classifica-regionale-lombardia.png`
- `2026-04-22_paesaggio_dolomiti-alba.mp4`
- `2026-04-23_demo_lifeline-inactivity-alert.mov`
- `2026-04-24_ambassador_marco-cuneo_trail.jpg`

Questo permette al social-manager di cercare asset rilevanti per tema/data.
