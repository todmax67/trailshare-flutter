# 📁 docs/gtm/ — Go-to-Market playbook

Cartella per materiali **operativi** di acquisition, outreach,
posizionamento e compliance legale dei flussi commerciali.

I documenti qui sono testi pronti da usare (template email,
copy legale, script vocali). La strategia di alto livello sta in:

- `ROADMAP.md` (root) — sezioni Epic 7 + Sprint
- `docs/B2B_PRICING.md` — pricing pubblico
- `docs/SEED_CLIENTS_OUTREACH.md` — programma Design Partner Pro free 6 mesi

## File in questa cartella

| File | Cosa | Quando usarlo |
|---|---|---|
| `PRESEEDING_PRIVACY_UPDATE.md` | Sezioni da aggiungere a Privacy Policy + Terms per supportare il pre-seed Spazi Pro (Epic 7.H) | Prima di mettere in prod il primo batch di Spazi Pro `unclaimed` |
| `OUTREACH_EMAILS_PRESEED.md` | 4 template email outreach per visite fisiche/email a rifugi pre-popolati | Dopo aver pre-popolato la regione target. Una variante per tipologia (rifugio, noleggio, guida, generic) |
| `OUTREACH_PHONE_SCRIPT.md` | Script telefonico/visita fisica 30s + 60s | Visite fisiche stagione apertura rifugi (giugno-luglio) |

## Convenzioni

- Tutti i template sono in **italiano formale-cordiale** (non commerciale aggressivo)
- Placeholder `{NOME_RIFUGIO}`, `{REGIONE}`, `{LINK_PAGINA_PUBBLICA}` da sostituire prima dell'invio
- Le citazioni GDPR fanno riferimento al **Regolamento UE 2016/679** + Codice Privacy italiano (D.Lgs. 196/2003 aggiornato 2018)
- Le email sono pensate per essere **personalizzate prima dell'invio** (mai inviare il template tale quale)

## Quando aggiornare

- **Privacy template**: ogni cambio di base legale o data source (es. aggiunta scraping di registro imprese richiede update)
- **Email template**: dopo ogni round di outreach, prendere il template della variante che ha funzionato meglio (open/reply rate) e fissarlo come default
- **Phone script**: dopo le prime 5 visite fisiche, raffinare con le obiezioni reali sentite
